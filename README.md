Clone o projeto localmente:
```cmd
git clone https://github.com/Nykcolas/meu_gestor_app.git
```

Acesse a pasta do projeto:
```cmd
cd meu_gestor_app
```

instale o composer no projeto
```cmd
composer install
```

Instale as dependências
```cmd
npm install
```

Instale as dependências
```cmd
npm run watch
```
